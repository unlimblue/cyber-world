# TOOLS.md - 本地笔记

技能定义工具 _如何_ 工作。这个文件是给你 _特定_ 的东西 — 你的设置独有的东西。

## 这里放什么

类似这样的东西：

- 摄像头名称和位置
- SSH 主机和别名
- TTS 偏好声音
- 扬声器/房间名称
- 设备昵称
- 任何环境特定的内容

## 示例

```markdown
### 摄像头

- living-room → 主区域，180° 广角
- front-door → 入口，动作触发

### SSH

- home-server → 192.168.1.100, 用户: admin

### TTS

- 偏好声音: "Nova" (温暖，略带英式)
- 默认扬声器: 厨房 HomePod
```

## 为什么分开？

技能是共享的。你的设置是你的。把它们分开意味着你可以更新技能而不会丢失你的笔记，分享技能而不会泄露你的基础设施。
